# Teacher — 把網頁變成凱比 App

講師（teacher）
APK 檔名：TeacherApp.apk

## 你的任務

把這個網站 → **https://allergy.4impact.cc** → 包成 Android App，裝到凱比上跑。

---

## 步驟

### 1. 進來就會看到 codex 在等你

你 attach 進 tmux 後，畫面會直接顯示：

```
>_ OpenAI Codex (v...)
model:     gpt-5.3-codex
directory: ~/workspace/liyu/codex/0
```

不用打任何指令，就可以直接「許願」。

### 2. 用「許願」請 codex 幫你做

把下面這段話貼進去：

```
請幫我把這個 Android App 專案打包成 APK，
取名叫 TeacherApp.apk，
然後上傳到老師的雲端。
做完跟我說一聲。
```

### 3. 等 codex 做事

codex 問「要不要執行這個指令？」 → 回 **Y**

第一次大概 30 秒〜1 分鐘。

### 4. 跟老師說「我上傳好了」

老師會把 APK 拉到凱比裝起來。

---

## 許願可以怎麼變？（你可以改！）

範本只是參考。下面這些都是你可以加進許願的句子：

- 「順便把 App 的名字改成 我的_過敏原小幫手」
- 「順便把 App 主色換成藍色」
- 「順便在登入畫面加一句『歡迎光臨！』」
- 「順便把 icon 換成這張圖：~/我的圖.png」

加好之後，再貼給 codex，它會一條一條幫你做。

---

## 卡住怎辦？三步驟口訣

1. 看清楚錯誤訊息
2. **整段複製**貼回給 codex
3. codex 會自己看怎麼修

不要關 terminal、不要慌。
還是不會 → **舉手叫老師**。
